# V3.8 Mobile Navigation & Layout Polish

ไฟล์นี้อธิบายการปรับแต่งสำหรับมือถือในเวอร์ชันล่าสุด

## ปรับเมนู 3 ขีด
- ใช้ปุ่มแบบ glass/gradient เข้ากับโลโก้
- แสดงเส้นเมนูชัดเจนและมีคำว่า “เมนู”
- เมื่อเปิดเมนู เส้นจะเปลี่ยนเป็น X

## ปรับเมนูมือถือ
- เมนูเปิดเป็น panel แบบมืออาชีพ
- เลื่อนได้เมื่อรายการเยอะ
- เมนูหลักเรียงเป็นการ์ด 2 ช่อง
- เมนูที่มีรายการย่อยขยายเต็มความกว้าง เพื่ออ่านง่าย

## ปรับการจัดเรียงข้อมูลบนมือถือ
- การ์ดสถิติ/จุดเด่น/ขั้นตอนบางส่วนจัดเป็น 2 ช่อง
- สลับการจัดวางซ้าย-ขวาใน Product / Project / Fleet
- จอเล็กมากยังคงบังคับให้อ่านง่าย ไม่ล้นจอ

## เหมาะสำหรับ
- ผู้สูงอายุ
- ลูกค้าที่เปิดผ่านมือถือ
- โทรติดต่อรวดเร็ว
- ดูผลงานและบริการได้เป็นระเบียบ
