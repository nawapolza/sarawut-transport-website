# หมายเหตุเวอร์ชันมือถือ V3.7

เวอร์ชันนี้ปรับให้เว็บไซต์ใช้งานบนมือถือได้ดีขึ้น โดยเฉพาะลูกค้าผู้สูงอายุและผู้ใช้งานทั่วไป

## สิ่งที่ปรับเพิ่ม

- เมนู 3 ขีดแบบใหม่ เปิดแล้วเลื่อนขึ้นลงได้ ไม่ล้นจอ
- ปุ่มโทรและปุ่มเพจเฟซบุ๊กใหญ่ขึ้น กดง่ายขึ้น
- เพิ่มแถบติดต่อด่วนด้านล่างมือถือ
- ปรับตัวอักษรและระยะห่างให้อ่านง่าย
- ปรับรูปภาพให้ไม่บีบ ไม่ล้นจอ และเรียงสวยบนมือถือ
- ปรับตารางข้อมูลบริษัทให้อ่านง่ายบนจอเล็ก
- รองรับจอมือถือเก่า เช่น 320px / 360px และมือถือรุ่นใหม่
- ลดปัญหาคอนเทนต์ล้นซ้ายขวา
- เพิ่ม safe-area สำหรับมือถือที่มีขอบจอ/รอยบาก

## วิธีทดสอบ

หลังรันเว็บแล้ว เปิด DevTools > Toggle Device Toolbar แล้วทดสอบขนาด:

- 320px
- 360px
- 390px
- 430px
- iPhone / Android รุ่นใหม่
- Tablet
- Desktop
