# Mobile Card Layout V3.9

เวอร์ชันนี้ปรับมือถือให้เป็น Mobile-first, Card-based มากขึ้น

- ปุ่มเมนู 3 ขีดทำใหม่ให้ดูพรีเมียมและเข้ากับโลโก้
- เมนูมือถือเป็นการ์ด 2 คอลัมน์สำหรับเมนูหลักที่สั้น
- เมนูที่มีรายการย่อยขยายเต็มแถวและเลื่อนขึ้นลงได้
- รูปแกลเลอรีบนมือถือเป็น 2×2 แบบมืออาชีพ
- การ์ดคู่ เช่น สถิติ ช่องทางติดต่อ ใช้ 2 คอลัมน์เฉพาะส่วนที่เหมาะสม
- การ์ดยาวยังคง 1 คอลัมน์เพื่อไม่ให้อ่านยาก
- รูปที่มีพิกัด GPS ถูกตัดส่วนพิกัดออกแล้ว
